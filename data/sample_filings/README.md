# Sample SEC Filing Data

This directory contains pre-processed excerpts from public SEC filings used for demo purposes.

The `sec_downloader.py` module can also fetch live filings from EDGAR at runtime.

## Data Source
All data is sourced from the SEC EDGAR public database (https://www.sec.gov/edgar).
